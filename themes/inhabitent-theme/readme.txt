#Inhabitent Wordpress Site

##Summary


A custom WordPress theme for an fictional camping website built using PHP, JQuery, and CSS/SASS. The about page is a custom template page which allows the client to modify the banner image in the Wordpress Backend. It includes one custom post type, one custom widget, one custom taxonomy, and a custom plugin built by me.

##Languages and tools used:


WordPress Content Mangement System
WordPress Underscore (theme skeleton)
Git/Github
JavaScript/JQuery
CSS/SASS
PHP
